## Screenshot Guidelines

1. **Resolution**: Minimum 1280x720, prefer 1920x1080
2. **Format**: PNG for static images, GIF/MP4 for animations
3. **Privacy**: Redact any personal information, student PII, or API keys visible
4. **Naming**: Use lowercase with hyphens, descriptive names
5. **Dark mode**: Prefer dark mode screenshots for consistency

## Tools for Capturing

- **macOS**: Cmd+Shift+4 (selection) or Cmd+Shift+5 (screen recording)
- **GIFs**: Use Gifox, LICEcap, or Kap
- **Screen recording**: QuickTime, OBS
- **Annotation**: Skitch, CleanShot X
